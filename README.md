# Power Clean 📚

도서 추천/리뷰 서비스 - 모노레포

## 프로젝트 구조

```
power-clean/
├── server/    # 백엔드 서버 (Spring Boot / Kotlin)
├── client/    # 프론트엔드 클라이언트 (React)
├── admin/     # 관리자 페이지
└── README.md
```
